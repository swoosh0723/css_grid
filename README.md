# CSS Grid
## grid란?
CSS Grid(그리드)는 2차원(행과 열)의 레이아웃 시스템을 제공합니다.



## 사용법
- 원하는 타입의 매크로 북마크로 변환
- 용산 IMAX, 용산 4DX
- 원하는 날짜, 시간 선택 후
- 북마크 클릭

## 북마크 타입 기준
- 시점
  1. 오픈
  2. 줍줍타임(무조건 1인)
- 상영관
  1. 용산 IMAX
  2. 용산 4dx
- 인원
  1. 1인
  2. 2인
